Introduction

안녕하세요, 개발자 안제호입니다!

끊임없이 도전하며 성장하는 풀스택 개발자입니다.

Java와 Spring Framework를 기반으로 한 웹 애플리케이션 개발 경험과 React를 활용한 프론트엔드 설계 역량을 갖추고 있습니다.

프로젝트를 통해 사용자 중심의 UI/UX와 안정적인 백엔드 시스템을 구현하며, AWS와 Docker를 활용한 배포 경험도 보유하고 있습니다.

현재 Python과 Next.js를 공부하며 새로운 기술을 습득하고 있습니다.

새로운 기술과 문제 해결을 통해 함께 성장할 수 있기를 기대합니다.

My Skills.

Program Language

객체지향 설계 및 컬렉션 프레임워크 활용, Stream API를 통한 데이터 처리 최적화, Spring Framework를 사용한 백엔드 개발 경험이 있습니다.

DOM 조작, 이벤트 처리, AJAX 비동기 통신 구현 경험이 있으며, jQuery와 fetch API를 활용한 데이터 처리와 상호작용을 구현했습니다.

Next.js와 React 기반 프로젝트에서 TypeScript를 활용하여 컴포넌트 설계 및 API 타입 정의를 경험했습니다.

Flask를 활용한 웹 애플리케이션 개발 경험이 있으며, Pandas를 통한 데이터 분석 및 데이터 처리를 구현했습니다.

Framework & Library

Spring MVC 기반의 백엔드 개발, 데이터베이스 연동, 트랜잭션 관리와 AOP 를 활용한 로깅 및 예외 처리에 익숙합니다.

React를 사용한 SPA 설계 경험이 있으며, Redux로 상태 관리와 컴포넌트 기반 UI 설계에 익숙합니다. RESTful API와 통합된 데이터 시각화도 가능합니다.

Contact Form 구현 및 동적 메시지 처리, Flask-WTF로 로그인 폼 생성 및 유효성 검증 기능을 적용한 경험이 있습니다.

간단한 책 목록 관리 애플리케이션을 개발하여, App Router 방식으로 폴더 기반 라우팅과 RESTful API를 구현하고, 책 추가, 수정, 삭제, 검색 등의 CRUD 기능을 제공한 경험이 있습니다.

DOM 조작, 이벤트 처리, AJAX 비동기 통신, JSON 데이터 처리, 애니메이션 효과, 폼 검증, UI 컴포넌트 제어, 그리고 Cookie 및 Session 관리를 통해 동적 웹 애플리케이션을 구현했습니다.

DBMS

정규화와 트랜잭션 관리를 통해 데이터 무결성을 유지하고 복잡한 쿼리를 최적화한 경험이 있습니다.

테이블 설계와 데이터 관리에 중점을 두고, DDL과 DML로 데이터를 효율적으로 관리한 경험이 있습니다.

UI/UX & MarkUp Language

웹 페이지 구조 설계와 콘텐츠 포지셔닝 작업이 어느정도는 가능합니다.

반응형 디자인, 레이아웃 최적화, 시각적 완성도를 높인 UI/UX 구현 경험이 있습니다.

반응형 네비게이션 바, 캐러셀, 가격 테이블 등 직관적인 UI 컴포넌트를 개발한적이 있습니다.

Cloud & Deployment

Docker를 활용한 컨테이너 기반 환경 구성 및 배포와 Spring MVC 프로젝트의 CI/CD 파이프라인 설계 경험이 있습니다.

AWS EC2와 RDS를 이용한 애플리케이션 배포 및 관리, RDS 마이그레이션, S3를 통한 정적 웹 호스팅과 백업 관리, Route53을 활용한 도메인 설정 및 DNS 관리 경험이 있습니다.

Tools & Development Environment

GitHub를 통해 코드 버전 관리 및 협업 프로젝트를 효율적인 형상관리를 통해 진행한 경험이 있습니다.

Postman을 통해, REST API를 테스트하며, 요청 및 응답을 검증이 가능합니다.

VSCode를 활용해 다양한 작업과 디버깅을 수행하며 효율적으로 프로젝트를 완성했습니다.

Jupyter Notebook으로 데이터 분석과 시각화를 수행한 경험이 여러번 있습니다.

PyCharm에서 Python 개발 환경을 구축하고 디버깅과 콘솔 사용법등을 익혀서 효율적인 코딩을 하였습니다.
